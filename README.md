**如何开始Hexo?**

1. 安装Nginx
    
    ```bash
    sudo  apt install nginx -y

    nginx -v
    ```
2. 安装Git

    git一般系统自带

    ```bash
    sudo  apt install git -y

    git --version
    ```
    > git version 2.25.1
    
3. 安装Node.js

    ```bash
    sudo apt install nodejs

    node -v
    ```

    > v10.19.0

    ```bash
    sudo apt install npm

    npm -v
    ```

4. 安装Hexo

**Hexo操作备忘录**

    ```bash
    hexo clean
    ```
