### 位操作

移位

```c
uint8_t a = 0x3C;	//a = 0b0011_1100
// 左移2位
uint8_t b = a << 2;	//b = 0b1111_0000
// 右移3位
uint8_t c = a >> 3;	//c = 0b0000_0111
```

取反

```c
uint8_t a = 0x3C;	//a = 0b0011_1100
uint8_t b = ~a;		//b = 0b1100_0011
```

位与

```c
uint8_t a = 0x3C;	//a = 0b0011_1100
uint8_t b = 0x68;	//b = 0b0110_1000
uint8_t c = a & b;	//c = 0b0010_1000
```

位或

```c
uint8_t a = 0x3C;	//a = 0b0011_1100
uint8_t b = 0x68;	//b = 0b0110_1000
uint8_t c = a | b;	//c = 0b0111_1100
```
